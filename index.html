Fanime

Fanime utilizes Kitsu, a third party API which houses a vast library of over 11,000 anime. Users can filter their genre of choice at the top of the Anime Library page. Also, by clicking the RANDOM ANIME button in the navigation menu, Fanime will randomly generate a title from Kitsu’s library where you can read the show’s synopsis. Every click returns a new anime that you can favorite or skip on to the next one. Favoriting an anime will give you access to it in your profiles favorite list. Fanime was built using Python/Django, PostgreSQL, HTML, CSS and Materialize.
